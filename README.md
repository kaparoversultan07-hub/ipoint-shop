 Интернет-магазин iPoint
Учебный проект интернет-магазина, выполненный в рамках стажировки Backend Developer.
 Реализованный функционал:
Каталог: Список категорий, пагинация товаров, фильтрация по цене и наличию.
Товары: Детальная карточка товара с остатком и описанием.
Корзина: Добавление/удаление товаров, изменение количества, расчет суммы.
Заказы:Оформление заказа, выбор способа оплаты и доставки.
Админ-панель:Просмотр списка заказов и смена их статусов.
 Инструкция по запуску:
1. Клонировать проект:
   `git clone https://github.com/kaparoversultan07-hub/ipoint-shop.git`
2. Перейти в папку: `cd ipoint-shop`
3. Установить зависимости: `composer install`
4. Создать файл настроек: `cp .env.example .env`
5. Сгенерировать ключ: `php artisan key:generate`
6. Запустить миграции и сидеры: `php artisan migrate --seed`
